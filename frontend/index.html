How to Use the App

Enter a Hex color code or RGB values to convert between them. Click the "Convert" button to see the results.

Hex to RGB Converter

- Converter tool. Converting HEX to RGB. Example format of HEX value: #ff0000

- Tool not saving any data

RGB to HEX Converter

- Converter tool. Use Values from 0 - 255 in each input box to get HEX value.

- Convert tool also save RGB and HEX value to data base. Data can be accessed with GET Values From Database form.

RGB to CMYK Converter

- Converter tool. Use Values from 0 - 255 in each input box to get CMYK value.

- Convert tool also save RGB and CMYK value to data base. Data can be accessed with GET Values From Database form.

CMYK to RGB Converter

- Converter tool. Use Values from 0 - 100 in each input box to get RGB value.

Get Values From Database

- Get values from database. Data is saved to database when using RGB to HEX.

- Data can be downloaded as .csv file.
